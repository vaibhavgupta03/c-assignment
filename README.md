# C Programming Assignments

This repository contains my solutions to various C programming assignments. The purpose of this folder is to practice and revise fundamental concepts in C programming, including data structures, algorithms, and problem-solving techniques.


## How to Run

To compile and run any C program, use the following commands in a terminal:

```bash
gcc filename.c -o output.exe
./output.exe
```

Replace `filename.c` with the actual C file name.

I am solving these coding problems and uploading the code to my GitHub repository to revise and practice C programming concepts. This helps in reinforcing learning and tracking progress over time.
